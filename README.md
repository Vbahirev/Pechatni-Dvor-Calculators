# Печатный Двор — платформа калькуляторов (Glass UI)

## Запуск
Откройте `index.html` (желательно через локальный сервер):
- VS Code → Live Server
- или `python -m http.server`

## Структура
- `core/` — общее ядро: тема, UI-хелперы, хранилище, движок расчётов
- `calculators/<id>/` — отдельные калькуляторы (каждый в своей папке)

## Добавить новый калькулятор
1) Создайте папку `calculators/<new-id>/`
2) Скопируйте шаблон из `calculators/laser/` и измените:
   - `config.json` (материалы, тарифы, доп.услуги)
   - `ui.html` (разметка)
   - `logic.js` (если нужна кастомная логика)
3) Добавьте запись в `main.js` в массив `calculators`

## Данные
- База (материалы/тарифы/услуги) хранится в `localStorage` по ключу `pd_data_<calcId>`
- Текущий расчёт (детали/галочки/наценка/клиент) — `pd_ui_<calcId>`
